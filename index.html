In this article, I want to talk about image hover effects. They may seem really complicated at first, but they're actually quite inuitive.

The CSS properties that are going to come in handy here are opacity and transform. What you do is, you say transition: amount-of-time ease; and opacity: 0; in a CSS class. Then, for a different class you write .that_class:hover { opacity: 1; } It also helps to use transform property, though it is not necessarily required.
